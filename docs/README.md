## PDF 만들기

`SETUP_GUIDE.html`은 인쇄 최적화된 CSS(`@media print`)를 포함하고 있어 브라우저에서 바로 PDF로 저장할 수 있습니다.

### Chrome/Edge에서
1. `SETUP_GUIDE.html` 파일을 브라우저에서 열기 (더블클릭)
2. `Ctrl + P` (인쇄 대화상자)
3. **대상**: "PDF로 저장" 선택
4. **용지**: A4
5. **여백**: 기본
6. **옵션**: "배경 그래픽" 체크 (색상·배경이 안 나오면)
7. 저장

### Safari에서
1. HTML 파일 열기
2. `⌘ + P` → 좌하단 `PDF` → "PDF로 저장"

## 배포 시 활용

- **학교에 배포할 때**: PDF를 이메일 첨부 or 인쇄본 우편 발송
- **원격 지원 시**: HTML 링크로 전달 (스크린 공유하며 안내)
- **QR 배포**: 앱 URL만 QR로 만들어 종이에 인쇄 (본 가이드 참고용으로 함께 배포)
